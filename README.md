The research outputs associated with this work are made available through multiple public GitHub repositories. The first repository, **OldSlavicLemma**, provides the implementation of the proposed neural lemmatization model, together with training, prediction, configuration, and experimental scripts:


https://github.com/usmannawaz01/OldSlavicLemma


The second repository, **OCS Lemmatization Benchmark**, provides the newly annotated Old Church Slavonic benchmark material, evaluation settings, preprocessing notes, result files, and scripts:


https://github.com/usmannawaz01/ocs-lemmatization-benchmark


Together, these repositories support the reproducibility of the work experiments and provide supplementary material for reported work. Dataset files are included or linked according to the redistribution permissions of the original sources.
